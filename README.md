Install bun on your system
Linux: ```curl -fsSL https://bun.sh/install | bash```
Windows: ```powershell -c "irm bun.sh/install.ps1 | iex"```

Then you run:
```bun install``` 
```bun run dev ``` (spins up a dev server)
or to build and run
```bun run build```
```bun run preview```
